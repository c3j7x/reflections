git remote (to check if there is a remote currently set up with the repository)
git remote add [RemoteName] [RemoteURL](to add a remote to the current repository and name it)
git remote -v (show the url where I will fetch data from and the url I will push data to)
git push [RemoteName] [Branch] (push local branch to remote)
git pull [RemoteName] [Branch] (pull from remote to local branch)
